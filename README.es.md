# Análisis de Datos Musicales 🎵

📘 También disponible en [inglés](./README.md)

Este repositorio contiene **ejercicios y proyectos de análisis de datos msuicales**. El objetivo es practicar la manipulación, exploración y visualización de datos musicales, incluyendo información sobre canciones, artistas, géneros y tendencias de reproducción.

----------

## 📦 Librerías utilizadas

Algunos de los paquetes más utilizados en los ejercicios son:

-   `pandas` – manipulación y análisis de datos
    
-   `numpy` – operaciones numéricas
    
-   `matplotlib` – visualización básica
    
-   `seaborn` – visualización estadística
    
-   `plotly` – visualizaciones interactivas (opcional)
    

----------

## 💻 Cómo usar este repositorio

1.  **Clonar el repositorio** en tu equipo:
2. 


    git clone https://github.com/nuriadevs/music-data-analysis.git 
    
    cd music-data-analysis



3.  **Crear un entorno virtual** (recomendado):
    

`python -m venv env  # Windows  env\Scripts\activate # Mac/Linux  source  env/bin/activate` 

3.  **Instalar las dependencias**:
    

`pip install -r requirements.txt` 

4.  **Abrir los notebooks**:
    

`jupyter notebook`
